    Руководство пользователя:

Данное приложение позволяет массово переименовывать
файлы с использованием шаблонов для поиска и
переименования файлов.

Правила для ввода шаблонов:
    - Шаблоны вводятся после перехода в опцию
    "Ввести шаблоны";
    - Шаблоны для поиска и переименования вводятся
    друг за другом и разделяются символом двоеточия;
    - В одной строке может быть только один символ
    двоеточия;
    - Слева и справа от символа двоеточия обязательно
    должны быть корректно введены шаблон для поиска и
    переименовывания соответственно;
    - В одной строке для ввода может быть только одна
    строка с шаблонами;
    - Перед первым шаблоном, между шаблонами и
    двоеточием и в конце строки может быть введено
    произвольное количество пробелов;
    - Максимальная длина строки с шаблонами ограничена
    размерами окна для ввода;
    - Слева от каждого некорректного шаблона ставится
    крестик, такой шаблон не обрабатывается;
    - Между началом/концом шаблона и точкой или между
    двумя точками в шаблоне может быть не более одного
    символа '*';
    - Символ '*' в шаблоне не должен стоять рядом с
    символом '?';
    - Недопустим ввод следующих символов: /<>|«;
    - При вводе пустой строки происходит выход из окна
    для ввода шаблонов.
Обработка шаблонов:
    - Символ '*' заменяется произвольным количеством
    любых других символов;
    - Символ '?' заменяется любым одним символом.
Примеры коректных шаблонов:
    - *.jpg : *.jpeg - Изменение расширения файлов с
    jpg на jpeg. image.jpg : image.jpeg;
    - hell*.txt : hello*.txt - 
    hellabc.txt : helloabc.txt;
    - hell*.txt : hell*o.txt - 
    hellabc.txt : hellabco.txt;
    - coo?.txt : coo?o.txt - К концу имени файла
    добавляется символ 'o'. cool.txt : coolo.txt;
    - *.txt : ????.txt - Имя файлов сокращается до
    первых 4 символов. geometry.txt : geom.txt;
    - hell*.txt : hell?.txt -
    hellooooo.txt : hello.txt.
После того, как шаблоны были введены и окно для ввода
закрыто, корректные шаблоны сохраняются в памяти
программы, а некорректные удаляются. При повторном
открывании окна для ввода шаблонов происходит вывод
всех корректных шаблонов.

Выбор каталога:
    - Данная опция позволяет выбрать каталог, в
    котором будет происходить поиск и переименование
    файлов;
    - Выбор каталога не происходит рекурсивно, то есть
    выбирать можно только из тех каталогов, которые
    расположены непосредственно в текущей директории.
После выбора каталога, выбранный каталог будет
сохранён и информация об этом будет видна в нижнем
левом угле интерфейса приложения.

Для запуска переименования нужно выбрать опцию
"Запустить массовое переименование". После этого
откроется новое окно, в котором будет выведена
информация о переименованных файлах. После завершения
операции, введённые шаблоны будут удалены.

Пункт "опции" позволяет более тонко настроить процесс
переименования файлов. В этом пункте меню можно
выбрать регистр, который будет у файлов после процесса
переименования. Данная опция будет применена для всех
файлов, подлежащих переименованию. Следующий параметр
позволяет вернуть приложение к заводским настройкам.
При его активации произойдёт удаление всех введённых
шаблонов, выбранный каталог вернётся к текущему, а
регистр вернётся к стандартному значению.

При активации пункта "Выход из приложения" происходит
выход из приложения.
